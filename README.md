### 🚀 Featured Projects

- **[Kadal Vizhi – Underwater Fish Tracking System](https://github.com/Vijayakumar-ece/Fish-Tracking)**
  AI-based system to detect and track fish in underwater videos using Python, OpenCV, YOLOv8, ByteTrack/BoT-SORT.

- **[ViStara – QR-Based Attendance System](https://github.com/Vijayakumar-ece/ViStara)**
  Web-based attendance management prototype built with HTML, CSS, and JavaScript for hackathons and demo pitches.

- **[Automatic Yarn Cone Sorting System](https://github.com/Vijayakumar-ece/Automatic-Yarn-Sorting)**
  Automated system to measure and sort yarn cones by weight using Arduino, Load Cell, HX711, and Servo Motor.

- **[Digital Voltage Meter](https://github.com/Vijayakumar-ece/Smart_Energy_Meter_Using_Arduino)**
  Digital voltage meter to measure and display DC voltage using Arduino Uno and a Voltage Sensor Module.

- **[Home Automation using LPC](https://github.com/Vijayakumar-ece/Home-automation-mini-project-using-lpc)**
  Bluetooth home automation using LPC2129 and HC-05 module to control lights and fans via UART commands.
